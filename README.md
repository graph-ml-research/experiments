# Эксперименты
Этот репозиторий содержит экспериментальную работу исследователей, не предназначенную для продуктива. 

## Типы файлов
Поддерживаются только текстовые файлы .txt, .py, .md$, .csv, .tsv, .json, .ya?ml. Пожалуйста, избегайте коммитов больших файлов в репозиторий.

## Язык сообщений коммитов

В этом репозитории сообщения коммитов могут быть написаны на русском или английском языке.
Использование русского языка допустимо для исследовательской и экспериментальной работы.

При этом сообщения коммитов должны:
- быть осмысленными и техническими;
- описывать, какие изменения были внесены и с какой целью;
- избегать неформальных или заглушечных формулировок.

Для репозиториев продуктива (`graph-ml`, `incubator`)
сообщения коммитов должны быть написаны на английском языке.

# Experiments
Experiments with graph machine learning methods, not ready for production. 

## File Types

Only text files are supported: .txt, .py, .md, .csv, .tsv, .json, .ya?ml. Please avoid committing large files to the repository.

## Commit language

In this repository, commit messages may be written in Russian or English.
Russian is acceptable for exploratory work.

However, commit messages must:
- be meaningful and technical
- describe what was changed and why
- avoid informal or placeholder messages

For production-ready repositories (`graph-ml`, `incubator`),
commit messages must be written in English.
